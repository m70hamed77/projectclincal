## ✅ المشروع جاهز! ابدأ الآن:

---

## 1️⃣ تشغيل المشروع:

### على هذا النظام (Linux/Bun):
```bash
cd /home/z/my-project
bun run dev
```

### على Windows:
- افتح `START_WIN.bat`
- أو شغّل:
```powershell
cd G:\smiley-clinc
npm run dev
```

---

## 2️⃣ افتح المشروع:

- **المتصفح:** `http://localhost:3000`
- **أو:** استخدم Preview Panel على اليمين

---

## 3️⃣ تسجيل الدخول:

### حساب Admin:
```
البريد: admin@smileydental.com
كلمة المرور: Admin@123456
```

### بعد الدخول:
- اذهب إلى `/admin` - لوحة تحكم الأدمن
- اذهب إلى `/dashboard/student` - لوحة الطالب
- اذهب إلى `/dashboard/patient` - لوحة المريض

---

## 4️⃣ المشاكل والحلول:

### السيرفر لا يعمل؟
```bash
# أوقف السيرفر القديم
pkill -f "next dev"

# شغّل السيرفر مرة أخرى
cd /home/z/my-project
bun run dev
```

### قاعدة البيانات لا تعمل؟
```bash
# إنشاء قاعدة البيانات
bun run db:push

# إنشاء Admin
bun run create-admin
```

### أخطاء في الكود؟
```bash
# التحقق من الأخطاء
bun run lint

# إصلاح تلقائي (إن وجد)
# قم بتعديل الملفات المطلوبة
```

---

## 5️⃣ الصفحات المتوفرة:

### الصفحات العامة:
- `/` - الصفحة الرئيسية
- `/auth/login` - تسجيل الدخول
- `/auth/register` - التسجيل الجديد

### صفحات الأدمن:
- `/admin` - لوحة التحكم الرئيسية
- `/admin/users` - إدارة المستخدمين
- `/admin/verification` - توثيق الطلاب
- `/admin/reports` - إدارة البلاغات

### صفحات الطالب:
- `/dashboard/student` - لوحة الطالب
- `/posts/create` - إنشاء حالة
- `/posts/my-posts` - حالاتي
- `/chat` - المحادثات

### صفحات المريض:
- `/dashboard/patient` - لوحة المريض
- `/posts` - عرض الحالات
- `/applications` - طلباتي

---

## 6️⃣ الأوامر المفيدة:

```bash
bun run dev              # تشغيل السيرفر
bun run lint             # التحقق من الكود
bun run db:push          # تحديث قاعدة البيانات
bun run db:studio        # فتح قاعدة البيانات
bun run create-admin     # إنشاء admin جديد
```
