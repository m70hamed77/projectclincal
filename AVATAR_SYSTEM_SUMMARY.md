## 🔌 API Endpoints

### 1️⃣ POST `/api/user/avatar`
**رفع صورة البروفايل**

**Headers:**
```
x-user-email: user@example.com
```

**Body (FormData):**
```
avatar: image file
```

**Response (Success):**
```json
{
  "success": true,
  "message": "تم رفع صورة البروفايل بنجاح",
  "avatarUrl": "/uploads/avatars/user_xxx_timestamp_random.jpg",
  "user": {
    "id": "user_id",
    "name": "User Name",
    "email": "user@example.com",
    "avatarUrl": "/uploads/avatars/user_xxx_timestamp_random.jpg"
  }
}
```

**Response (Error):**
```json
{
  "error": "يجب اختيار صورة صالحة (JPG, PNG, GIF, WebP)"
}
```

---

### 2️⃣ DELETE `/api/user/avatar`
**حذف صورة البروفايل**

**Headers:**
```
x-user-email: user@example.com
```

**Response (Success):**
```json
{
  "success": true,
  "message": "تم حذف صورة البروفايل بنجاح"
}
```

**Response (Error):**
```json
{
  "error": "لا توجد صورة بروفايل"
}
```

## 🧪 كيفية الاستخدام

### 1️⃣ رفع صورة جديدة
1. اذهب إلى صفحة الإعدادات
2. اضغط على "رفع صورة"
3. اختر صورة من جهازك
4. تظهر الصورة فوراً

### 2️⃣ تغيير الصورة
1. اضغط على "تغيير الصورة"
2. اختر صورة جديدة
3. الصورة القديمة تُحذف تلقائياً
4. الصورة الجديدة تظهر فوراً

### 3️⃣ حذف الصورة
1. اضغط على "حذف الصورة"
2. أكد الحذف
3. الصورة تُحذف من السيرفر والـ DB

### 4️⃣ عرض الصورة
- **صفحة البروفايل:** تظهر الصورة في الهيدر
- **صفحة الإعدادات:** تظهر الصورة في قسم صورة البروفايل
- **الـ Navigation:** تظهر الصورة في القائمة الجانبية
