# 📊 Real-Time Stats System

## 🧠 الفكرة الأساسية

النظام الآن يعتمد على **قاعدة البيانات الفعلية** وليس على عدادات يدوية. كل تسجيل أو إضافة في النظام ينعكس تلقائياً على إحصائيات الـ Dashboard.

---

## 🎯 الإحصائيات الأربعة الرئيسية

### 1. 🎓 عدد الطلاب (Students)

**مصدر البيانات:** `Student` Table

```sql
SELECT COUNT(*) FROM Student;
-- إجمالي الطلاب

SELECT COUNT(*) FROM Student
WHERE verificationStatus = 'APPROVED'
  AND user.status = 'ACTIVE';
-- الطلاب النشطين (الموثقين)
```

**متى يتحدث:**
- عند تسجيل طالب جديد → `+1`
- عند حذف طالب → `-1`
- عند تغيير حالة التحقق → يتحدث عدد الطلاب النشطين

---

### 2. 🧑‍⚕️ عدد المرضى (Patients)

**مصدر البيانات:** `Patient` Table

```sql
SELECT COUNT(*) FROM Patient;
-- إجمالي المرضى

SELECT COUNT(*) FROM Patient
WHERE user.status = 'ACTIVE';
-- المرضى النشطين
```

**متى يتحدث:**
- عند تسجيل مريض جديد → `+1`
- عند حذف مريض → `-1`
- عند تغيير حالة الحساب → يتحدث عدد المرضى النشطين

---

### 3. 🩺 عدد الحالات (Cases)

**مصدر البيانات:** `Case` Table

```sql
SELECT COUNT(*) FROM Case WHERE isCompleted = true;
-- الحالات المكتملة

SELECT COUNT(*) FROM Case WHERE isCompleted = false;
-- الحالات النشطة (قيد التنفيذ)
```

**متى يتحدث:**
- عند إنشاء حالة جديدة → `+1` في الحالات النشطة
- عند إكمال حالة → `+1` في المكتملة، `-1` في النشطة
- عند حذف حالة → `-1`

---

### 4. ⭐ التقييم (Ratings)

**مصدر البيانات:** `Rating` Table

```sql
SELECT COUNT(*) FROM Rating;
-- إجمالي التقييمات

SELECT AVG(overallRating) FROM Rating;
-- متوسط التقييم العام
```

**متى يتحدث:**
- عند إضافة تقييم جديد → `+1` في عدد التقييمات، إعادة حساب المتوسط
- عند تعديل تقييم → إعادة حساب المتوسط
- عند حذف تقييم → `-1` في عدد التقييمات، إعادة حساب المتوسط

### Auto-Refresh

النظام يقوم بتحديث الإحصائيات تلقائياً كل **10 ثواني**:

```typescript
useEffect(() => {
  const interval = setInterval(() => {
    if (user && user.role === 'ADMIN') {
      fetchStats()
    }
  }, 10000) // 10 seconds

  return () => clearInterval(interval)
}, [user])
```

## 🚀 كيفية الاستخدام

### للمطورين

1. **عند إضافة مستخدم جديد:**
   - لا تحتاج لتعديل أي عداد
   - النظام يحسب تلقائياً

2. **عند إنشاء حالة جديدة:**
   - فقط INSERT في جدول `Case`
   - الإحصائيات تتحدث تلقائياً

3. **عند إضافة تقييم:**
   - فقط INSERT في جدول `Rating`
   - النظام يحسب المتوسط تلقائياً

### للأدمن

1. **عرض الإحصائيات:**
   - افتح لوحة التحكم `/admin`
   - شاهد الإحصائيات الحية

2. **التحديث التلقائي:**
   - لا داعي لتحديث الصفحة
   - النظام يحدث كل 10 ثواني

3. **البيانات الدقيقة:**
   - كل رقم حقيقي من قاعدة البيانات
   - لا يوجد تقديرات أو تخمينات
