## 🛠️ إذا واجهت مشاكل (نادرة جداً):

### المشكلة: "port 3000 already in use"

**الحل 1:**
```bash
# أوجد العملية
lsof -ti:3000

# أوقفها
kill -9 <PID>
```

**الحل 2:**
```bash
# أعد تشغيل السيرفر في terminal جديد
Ctrl + C (في terminal القديم)
bun run dev
```

### المشكلة: "Cannot connect to database"

**الحل:**
```bash
# تحقق من الاتصال بالإنترنت
# NeonDB يحتاج اتصال بالإنترنت
```

### المشكلة: "Module not found"

**الحل:**
```bash
# أعد تثبيت الحزم
rm -rf node_modules
bun install
bun run dev
```

### المشكلة: الصفحة لا تفتح

**الحل:**
```bash
# 1. تأكد أن السيرفر يعمل
# يجب أن ترى: "✓ Ready in X.Xs"

# 2. افتح Developer Tools (F12)
# اقرأ رسالة الخطأ في Console

# 3. أعد تحميل الصفحة (Ctrl + Shift + R)
```
