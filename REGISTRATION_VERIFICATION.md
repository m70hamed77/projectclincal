## 📝 خطوات التسجيل التفصيلية

### الخطوة 1: البيانات الأساسية

**المستخدم يقوم بـ:**
1. اختيار نوع الحساب (طالب/دكتور أو مريض)
2. إدخال الاسم الكامل
3. إدخال البريد الإلكتروني
4. إدخال كلمة المرور
5. تأكيد كلمة المرور

**الزر: "إرسال كود التحقق"**

**شروط تفعيل الزر:**
- ✅ الاسم: 3 أحرف على الأقل
- ✅ البريد الإلكتروني: 6 أحرف على الأقل
- ✅ كلمة المرور: 6 أحرف على الأقل
- ✅ تأكيد كلمة المرور: متطابق مع كلمة المرور

**ما يحدث عند الضغط:**
1. استدعاء `/api/auth/send-verification-code`
2. توليد كود تحقق من 6 أرقام
3. حفظ الكود في الذاكرة
4. عرض الكود في Console (للتطوير)
5. الانتقال للخطوة 2

---

### الخطوة 2: التحقق من الكود

**المستخدم يقوم بـ:**
1. إدخال كود التحقق (6 أرقام)
2. الكود يظهر في Console للمطور

**الزر: "إنشاء الحساب" (للمرضى) أو "تحقق من الكود" (للطلاب)**

**شروط تفعيل الزر:**
- ✅ كود التحقق: 6 أرقام بالضبط

**ما يحدث عند الضغط:**

#### للمريض:
1. التحقق من صحة الكود
2. استدعاء `/api/auth/register-user` (JSON)
3. إنشاء المستخدم بحالة `ACTIVE`
4. إنشاء بروفايل Patient
5. التوجيه لصفحة تسجيل الدخول

#### للطالب:
1. التحقق من صحة الكود
2. الانتقال للخطوة 3 لإدخال البيانات الإضافية

### الخطوة 3: البيانات الإضافية (للطلاب فقط)

**ما يحدث عند الضغط:**
1. التحقق من جميع الحقول

## 🔧 تفاصيل الـ API

### `/api/auth/send-verification-code`

**الطريقة:** POST

**الجسم (Body):**
```json
{
  "email": "user@example.com"
}
```

**الاستجابة:**
```json
{
  "success": true,
  "message": "تم إرسال كود التحقق إلى بريدك الإلكتروني",
  "devCode": "123456" // يظهر في وضع التطوير فقط
}
```

---

### `/api/auth/register-user`

**الطريقة:** POST

#### للمريض (JSON):
```json
{
  "name": "الاسم الكامل",
  "email": "patient@example.com",
  "password": "password123",
  "phone": "01012345678",
  "role": "PATIENT"
}
```

#### للطالب (FormData):
```
name: "الاسم الكامل"
email: "student@example.com"
password: "password123"
phone: "01012345678"
universityEmail: "uni@example.com"
academicYear: "3"
carniehImage: [File]
role: "STUDENT"
```

**الاستجابة:**
```json
{
  "success": true,
  "message": "تم إنشاء حسابك بنجاح! يمكنك الآن تسجيل الدخول.",
  "user": {
    "id": "user-id",
    "name": "الاسم الكامل",
    "email": "user@example.com",
    "role": "PATIENT",
    "verificationStatus": null
  }
}
```
