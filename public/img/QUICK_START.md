# 🎯 البدء السريع - Quick Start Guide

## ✅ ما تم إعداده لك

### 1. 📁 مجلد الصور
```
public/img/
├── README.md           - معلومات أساسية
├── USAGE_GUIDE.md      - دليل شامل (اقرأ هذا!)
└── QUICK_START.md      - هذا الملف
```

### 2. 📄 الصفحات المحدثة
تم إضافة دعم خلفيات الصور في:

| الصفحة | الملف | مسار الصورة المقترح |
|--------|-------|---------------------|
| تسجيل الدخول | `/src/app/auth/login/page.tsx` | `/img/login-bg.jpg` |
| التسجيل | `/src/app/auth/register/page.tsx` | `/img/register-bg.jpg` |
| نسيت كلمة المرور | `/src/app/forgot-password/page.tsx` | `/img/forgot-password-bg.jpg` |

---

## 🚀 كيفية البدء (3 خطوات بسيطة)

### الخطوة 1: إضافة صورتك
انسخ صورتك واضعها في مجلد `public/img/`
```
my-image.jpg → public/img/my-image.jpg
```

### الخطوة 2: تحديث الكود
اذهب للملف المطلوب وابحث عن:
```tsx
backgroundImage: 'url("/img/...")',
```
غيّر اسم الملف لاسم صورتك

### الخطوة 3: احفظ وشاهد النتيجة! 🎉

---

## 📝 مثال عملي

### إضافة خلفية لصفحة تسجيل الدخول:

1. **ضع صورتك:**
   ```
   my-dental-clinic.jpg → public/img/my-dental-clinic.jpg
   ```

2. **عدّل الكود في `/src/app/auth/login/page.tsx`:**
   ```tsx
   // ابحث عن السطر رقم 319 تقريباً
   style={{
     backgroundImage: 'url("/img/my-dental-clinic.jpg")',  // ← غيّر هذا
   }}
   ```

3. **احفظ الملف وشاهد النتيجة!**

## 🎯 قائمة سريعة

- ✅ مجلد `public/img/` تم إنشاؤه
- ✅ صفحة تسجيل الدخول جاهزة للصورة
- ✅ صفحة التسجيل جاهزة للصورة
- ✅ صفحة نسيت كلمة المرور جاهزة للصورة
- ✅ تعليقات واضحة في الكود
- ✅ دليل شامل متوفر
- ✅ طبقات شفافة مضافة لضمان وضوح النصوص
