# مجلد الصور - Background Images Directory

هذا المجلد يحتوي على الصور المستخدمة كخلفيات في الموقع.

## كيفية إضافة صور جديدة:

1. ضع صورك المخصصة في هذا المجلد
2. استخدم المسارات التالية في الكود:
   - للصور في هذا المجلد: `/img/filename.ext`
   - مثال: `/img/login-bg.jpg`

## أسماء الصور المقترحة:

- `login-bg.jpg` - خلفية صفحة تسجيل الدخول
- `register-bg.jpg` - خلفية صفحة التسجيل
- `forgot-password-bg.jpg` - خلفية صفحة نسيت كلمة المرور
- `home-hero-bg.jpg` - خلفية القسم الرئيسي في الصفحة الرئيسية
- `dashboard-bg.jpg` - خلفية لوحة التحكم
- `default-bg.jpg` - خلفية افتراضية

## الصيغ المدعومة:

- `.jpg`, `.jpeg`
- `.png`
- `.webp`
- `.svg`

## نصائح:

- يفضل استخدام صور بحجم كبير (مثل 1920x1080 أو أكبر)
- استخدم ضغط جيد للأصوات لتسريع تحميل الموقع
- يفضل استخدام تنسيق WebP للحجم الأصغر
- يمكن استخدام الصور بألوان فاتحة مع شفافية لتبقى النصوص واضحة

---

This folder contains background images used throughout the site.

## How to add new images:

1. Place your custom images in this folder
2. Use the following paths in your code:
   - For images in this folder: `/img/filename.ext`
   - Example: `/img/login-bg.jpg`

## Suggested image names:

- `login-bg.jpg` - Login page background
- `register-bg.jpg` - Register page background
- `forgot-password-bg.jpg` - Forgot password page background
- `home-hero-bg.jpg` - Home page hero section background
- `dashboard-bg.jpg` - Dashboard background
- `default-bg.jpg` - Default background

## Supported formats:

- `.jpg`, `.jpeg`
- `.png`
- `.webp`
- `.svg`

## Tips:

- Use large images (e.g., 1920x1080 or larger)
- Use good compression for faster loading
- WebP format is recommended for smaller file size
- Use light colors with transparency to keep text readable
