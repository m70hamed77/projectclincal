# إعداد نظام الرسائل النصية (SMS) - Smiley Dental Clinic

## 📱 نظرة عامة

تم تفعيل نظام إرسال الرسائل النصية (SMS) في المشروع باستخدام خدمة **Twilio**. النظام يدعم الآن طريقتين للتحقق:
1. ✅ البريد الإلكتروني (Email)
2. ✅ الرسائل النصية (SMS)

## ✅ ما تم تنفيذه

1. ✅ تثبيت حزمة `twilio`
2. ✅ إنشاء ملف `src/lib/sms.ts` لإعداد نظام إرسال SMS
3. ✅ إنشاء API `/api/auth/send-sms-code` لإرسال كود التحقق عبر SMS
4. ✅ إنشاء API `/api/auth/verify-sms-code` للتحقق من كود SMS
5. ✅ تحديث صفحة التسجيل لإضافة خيار اختيار طريقة التحقق
6. ✅ إنشاء صفحة `/auth/verify-sms` للتحقق عبر SMS
7. ✅ تحسين API التحقق من الإيميل لإصلاح مشكلة الكود

## 🔄 الوضع الحالي

في وضع التطوير (Development Mode)، النظام:
- ✅ يعرض محتوى الرسالة في **console**
- ✅ يعرض الكود في console للمساعدة في الاختبار
- ⚠️ **لا يرسل** SMS فعلياً إلى الهاتف

## 🚀 كيفية تفعيل إرسال SMS الفعلي

لإرسال SMS فعلياً للمستخدمين، اتبع الخطوات التالية:

### 1. إنشاء حساب على Twilio

1. اذهب إلى: https://www.twilio.com/
2. اضغط على **"Sign Up"** واملأ البيانات
3. احصل على رصيد مجاني بقيمة **$15** (يكفي لإرسال الكثير من الرسائل)
4. أكمل عملية التحقق من البريد الإلكتروني والهاتف

### 2. الحصول على بيانات الحساب

1. بعد تسجيل الدخول، اذهب إلى: https://console.twilio.com/
2. اذهب إلى **Settings** > **General**
3. ستجد:
   - **Account SID**: تبدأ بـ `AC...`
   - **Auth Token**: اضغط على "Show" لرؤيته

### 3. الحصول على رقم هاتف Twilio

1. اذهب إلى: https://console.twilio.com/us1/develop/phone-numbers/manage/incoming
2. اضغط على **"Buy a number"**
3. اختر الدولة (مثلاً: Egypt أو United States)
4. اختر رقم من القائمة
5. اضغط **"Buy"**

### 4. إضافة البيانات إلى المشروع

افتح ملف `.env` في مجلد المشروع واستبدل:

```env
TWILIO_ACCOUNT_SID=your_account_sid_here
TWILIO_AUTH_TOKEN=your_auth_token_here
TWILIO_PHONE_NUMBER=your_twilio_phone_number
```

ببياناتك الحقيقية:

```env
TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TWILIO_AUTH_TOKEN=your_auth_token_here
TWILIO_PHONE_NUMBER=+1234567890
```

### 5. إعادة تشغيل السيرفر

```bash
# أوقف السيرفر الحالي (Ctrl+C)
# ثم أعد تشغيله
bun run dev
```

## 📨 كيفية الاستخدام

### للتسجيل عبر SMS:

1. اذهب إلى صفحة التسجيل: `/auth/register`
2. اختر النوع (مريض أو طالب)
3. املأ البيانات
4. في قسم "طريقة التحقق"، اختر **"رسالة نصية (SMS)"**
5. اضغط **"إرسال كود التحقق"**
6. أدخل الكود المستلم
7. اضغط **"تحقق"**

### في وضع التطوير (بدون بيانات Twilio):

1. افتح **Console** في المتصفح (F12)
2. ستجد سطر مثل:
   ```
   [SMS Verification] Code for 01234567890: 123456
   ```
3. سيتم عرض محتوى الرسالة في console:
   ```
   ============================================================
   [SMS SENDING SIMULATION - Development Mode]
   ============================================================
   To: +201234567890
   Message: كود التحقق من سمايلي لطب الأسنان: 123456
   ============================================================
   ```

### في وضع الإنتاج (مع بيانات Twilio):

1. سيتم إرسال SMS فعلياً إلى الهاتف
2. الرسالة ستكون باللغة العربية
3. الكود صالح لمدة 10 دقائق

## 🐛 استكشاف الأخطاء

### لم تصل الرسالة؟

1. **تحقق من Console**: قد يكون في وضع التطوير
2. **تأكد من بيانات Twilio**: Account SID و Auth Token يجب أن يكونوا صحيحين
3. **تأكد من رقم الهاتف**: يجب أن يكون بالتنسيق الصحيح (+20xxxxxxxxx)
4. **افحص الرصيد**: تأكد من وجود رصيد في حساب Twilio

### خطأ "Invalid Account SID"؟

1. تأكد من نسخ Account SID بالكامل (يبدأ بـ AC)
2. تحقق من أن المفتاح لا يزال فعالاً
3. تأكد من إضافته في ملف `.env`

### خطأ "Authentication Error"؟

1. تأكد من نسخ Auth Token بالكامل
2. اضغط "Show" لرؤية التوكن الكامل
3. تأكد من إضافته في ملف `.env`

### خطأ "From number is not verified"؟

1. في الحساب المجاني، يجب التحقق من الأرقام
2. اذهب إلى: https://console.twilio.com/us1/develop/phone-numbers/manage/verified
3. أضف رقمك للتحقق منه
4. أو استخدم رقم Twilio الذي اشتريته

## 📝 ملاحظات مهمة

1. **لا تلتزم ملف .env**: أضف `.env` إلى `.gitignore`
2. **تنسيق الأرقام**: النظام يقوم بتحويل الأرقام المصرية تلقائياً:
   - `01234567890` → `+201234567890`
3. **راقب الاستهلاك**: تأكد من عدم تجاوز الرصيد
4. **اختبر جيداً**: قبل الإطلاق، تأكد من أن الرسائل تصل بشكل صحيح

## 🔧 الدوال المتاحة

### 1. sendSMS()
إرسال رسالة نصية مخصصة

```typescript
import { sendSMS, formatPhoneNumber } from '@/lib/sms'

const formattedPhone = formatPhoneNumber('01234567890')
await sendSMS(formattedPhone, 'مرحباً بك في سمايلي!')
```

### 2. sendVerificationSMS()
إرسال كود التحقق

```typescript
import { sendVerificationSMS } from '@/lib/sms'

await sendVerificationSMS('01234567890', '123456')
```

### 3. formatPhoneNumber()
تنسيق رقم الهاتف

```typescript
import { formatPhoneNumber } from '@/lib/sms'

// 01234567890 → +201234567890
const formatted = formatPhoneNumber('01234567890')
```
