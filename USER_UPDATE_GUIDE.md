# 📝 دليل تحديث بيانات المستخدم

## 🎯 **الميزة الجديدة:**

إمكانية تعديل وحفظ بيانات الحساب الشخصي من صفحة الإعدادات.

---

## ✅ **ما يمكن تعديله:**

| الحقل | الوصف |
|-------|-------|
| الاسم الكامل | اسم المستخدم |
| رقم الهاتف | رقم الاتصال |
| النبذة التعريفية | وصف قصير عن المستخدم |

**⚠️ ملاحظة:** البريد الإلكتروني لا يمكن تعديله حالياً (لأسباب أمنية).

---

## 🚀 **كيفية الاستخدام:**

### **الخطوة 1: الذهاب إلى صفحة الإعدادات**
```
/settings
```

### **الخطوة 2: تعديل البيانات**
- اكتب اسمك الجديد في حقل "الاسم الكامل"
- اكتب رقم هاتفك الجديد في حقل "رقم الهاتف"
- اكتب نبذة عنك في حقل "نبذة تعريفية"

### **الخطوة 3: حفظ التغييرات**
- انقر على زر "حفظ التغييرات" 💾
- انتظر حتى تظهر رسالة النجاح ✅

### **الخطوة 4: التحقق**
- البيانات ستحفظ في قاعدة البيانات
- ستتحدث فوراً في جميع أنحاء التطبيق
- ستحفظ أيضاً في localStorage

---

## 🔧 **التقنية:**

### **Frontend:**
- React State Management
- Form Handling
- Console Logging للتشخيص

### **Backend:**
- API Endpoint: `/api/user/update`
- Database Update باستخدام Prisma
- Validation و Error Handling

### **Database:**
- تحديث حقول `name` و `phone` في جدول `User`

---

## 📊 **سير العمل:**

```
1. المستخدم يعدل البيانات في الـ Form
   ↓
2. ينقر على "حفظ التغييرات"
   ↓
3. Frontend يرسل البيانات إلى `/api/user/update`
   ↓
4. Backend يتحقق من البيانات
   ↓
5. يتم تحديث قاعدة البيانات
   ↓
6. Backend يرسل البيانات المحدثة
   ↓
7. Frontend يحدث localStorage و State
   ↓
8. تظهر رسالة نجاح ✅
```

---
