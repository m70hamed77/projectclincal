# 🚀 طريقة التشغيل الصحيحة على Windows

## الطريقة الصحيحة ⭐

### 1️⃣ افتح PowerShell واكتب:

```powershell
# عيّن المتغير أولاً
$env:TURBOPACK=0

# ثم شغل السيرفر
npm run dev
```

---

## ⚠️ ماذا تفعل الآن؟

### الخطوة 1: أوقف السيرفر الحالي

اضغط `Ctrl+C` في PowerShell

### الخطوة 2: شغل بهذه الطريقة

```powershell
# كتابة المتغير
$env:TURBOPACK=0

# شغل السيرفر
npm run dev
```

---

## ❓ كيف تعرف أن Turbopack معطّل؟

عند الشغيل، إذا رأيت:
```
▲ Next.js 16.1.6 (Webpack)
```
✅ يعني أنه يعمل بشكل صحيح

إذا رأيت:
```
▲ Next.js 16.1.6 (Turbopack)
```
❌ يعني أن Turbopack لا يزال يعمل

---

## 🔧 إذا استمرت المشكلة:

### الحل: احذف مجلد `.next` وأعد التشغيل

```powershell
# حذف مجلد .next
rmdir /s /q .next

# عيّن المتغير
$env:TURBOPACK=0

# شغل السيرفر
npm run dev
```

---

## ✅ بعد التشغيل الصحيح:

1. افتح المتصفح: `http://localhost:3000`
2. سجل الدخول باستخدام:
   - البريد: admin@smileydental.com
   - كلمة المرور: Admin@123456
3. إذا ظهر خطأ 500، **انسخ ما يظهر في Terminal وأرسله لي**

---

## 📝 ملاحظات مهمة:

- لا تستخدم `set TURBOPACK=0` في PowerShell (هذا للأمر CMD القديم)
- استخدم دائماً `$env:TURBOPACK=0` في PowerShell
- `$env:` يعتبر تعيين مؤقت للجلسة الحالية
